# Prompt Hub

一个提示词管理和搜索平台，支持多种数据源和智能搜索。

## 功能特性

- 🔍 智能搜索提示词
- 📁 多数据源支持（prompts、banana、gpt4o）
- 🏷️ 分类筛选（剧本、写作、营销、代码等）
- 📱 响应式设计
- 🎨 美观的卡片式展示
